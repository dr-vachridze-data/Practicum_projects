# Исследование рынка стартапов

## Описание проекта

В рамках проекта проведён анализ данных о финансировании стартапов с целью выявления перспективных отраслей для инвестирования и наиболее эффективных типов финансирования.

Исследование включает очистку данных, выделение рыночных сегментов, поиск аномалий, анализ инвестиционной активности и изучение динамики возврата инвестиций.

## Цель исследования

На основе исторических данных определить:

- перспективные направления для инвестирования;
- наиболее эффективные типы финансирования;
- тенденции развития рынка стартапов;
- сегменты с устойчивым ростом объёмов инвестиций.

## Задачи проекта

### Подготовка данных

- обработка пропусков и дубликатов;
- преобразование типов данных;
- восстановление недостающих дат финансирования;
- оценка качества данных.

### Формирование признаков

- группировка компаний по срокам финансирования;
- классификация рыночных сегментов на массовые, средние и нишевые;
- расчёт дополнительных аналитических показателей.

### Исследовательский анализ данных

- анализ распределения объёмов финансирования;
- выявление аномальных компаний методом IQR;
- исследование структуры рынка по сегментам;
- сравнение различных типов финансирования.

### Анализ динамики

- динамика среднего размера инвестиционного раунда;
- изменение активности рынка по годам;
- динамика финансирования по сегментам;
- анализ возврата инвестиций.

## Используемые технологии

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Основные методы анализа

- исследовательский анализ данных (EDA);
- создание новых признаков;
- выявление и обработка выбросов (IQR);
- анализ временных рядов;
- визуализация данных.

## Результаты

В результате исследования:

- выявлены сегменты рынка с положительной динамикой финансирования;
- определены наиболее популярные типы финансирования;
- проанализирована эффективность различных инвестиционных инструментов;
- сформированы рекомендации для потенциального инвестора.
