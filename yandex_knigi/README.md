# Анализ пользовательской активности и результатов A/B-тестирования

## Описание проекта

Проект посвящен проверке статистических гипотез и анализу результатов A/B-тестирования. В ходе работы были исследованы различия в пользовательской активности между двумя городами, а также проведена оценка эффективности нового интерфейса интернет-магазина на основе экспериментальных данных.

## Задачи проекта

Проект состоит из двух независимых исследований.

### Часть 1. Анализ пользовательской активности

Цель исследования — определить, отличается ли среднее время активности пользователей приложения в Москве и Санкт-Петербурге.

В рамках анализа были выполнены следующие этапы:

* загрузка и проверка подготовленных данных;
* поиск дубликатов по идентификаторам пользователей;
* сравнение размеров выборок;
* расчет описательной статистики;
* проверка статистической гипотезы с помощью одностороннего t-теста для независимых выборок;
* интерпретация полученного p-value и формулировка выводов;
* анализ возможных причин выявленных различий.

### Проверяемая гипотеза

**Нулевая гипотеза (H₀):** среднее время активности пользователей Санкт-Петербурга не превышает среднее время активности пользователей Москвы.

**Альтернативная гипотеза (H₁):** пользователи Санкт-Петербурга проводят в приложении больше времени, чем пользователи Москвы.

---

### Часть 2. Анализ результатов A/B-тестирования

Во второй части проекта проведена оценка результатов A/B-теста нового интерфейса интернет-магазина.

Цель эксперимента — определить, приводит ли упрощение интерфейса к увеличению конверсии зарегистрированных пользователей в покупателей в течение первых семи дней после регистрации.

В ходе анализа были выполнены:

* загрузка данных об участниках эксперимента и пользовательских событиях;
* проверка корректности проведения A/B-теста;
* анализ распределения пользователей по группам;
* исследование пользовательских действий;
* расчет конверсии для контрольной и тестовой групп;
* проверка статистической значимости различий между группами;
* оценка эффективности внедренного интерфейса.

## Используемые инструменты

* Python
* Pandas
* NumPy
* SciPy
* Matplotlib
* Seaborn

## Применяемые методы

* предобработка данных;
* исследовательский анализ данных (EDA);
* описательная статистика;
* проверка статистических гипотез;
* z-тест для сравнения долей;
* анализ конверсии;
* анализ результатов A/B-тестирования.

## Результат

В результате работы были проверены статистические гипотезы, оценена корректность проведения эксперимента и проанализировано влияние изменений интерфейса на поведение пользователей. Полученные результаты позволяют сделать вывод о наличии или отсутствии статистически значимых различий между исследуемыми группами.
