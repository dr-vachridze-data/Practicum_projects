# Исследование рынка видеоигр (2000–2013)

## Описание проекта

Проект посвящён исследованию рынка видеоигр в период с 2000 по 2013 год. Анализ выполняется для подготовки статьи о развитии игровой индустрии начала XXI века и изучения факторов, влияющих на популярность игр различных жанров и платформ.

Особое внимание уделяется жанру RPG (Role-Playing Games), а также региональным особенностям продаж и оценкам пользователей и критиков.

В рамках проекта проводится полный цикл подготовки данных: от первичного знакомства с датасетом до очистки, предобработки, фильтрации и категоризации данных.

---

## Цели исследования

* Изучить развитие игровой индустрии в период 2000–2013 гг.
* Проанализировать продажи игр на различных платформах.
* Исследовать распределение игр по жанрам.
* Изучить влияние пользовательских и критических оценок.
* Определить наиболее популярные игровые платформы.
* Подготовить качественный набор данных для дальнейшего анализа.

---

### 2. Предобработка данных

#### Приведение названий столбцов

Пример:

```text

```

#### Проверка типов данных

Проведена проверка соответствия типов данных содержимому столбцов.

При необходимости:

* строковые значения в числовых столбцах заменены на пропуски;

#### Обработка пропусков

Для каждого столбца рассчитаны:

В ходе анализа были:

* выбраны способы их обработки;
* пропуски в продажах при необходимости заменены средними значениями по платформе и году выпуска.

#### Работа с дубликатами

* приведение названий игр и жанров к нижнему регистру;
* приведение рейтингов ESRB к верхнему регистру;

После очистки подсчитано количество удалённых строк в абсолютных и относительных значениях.
